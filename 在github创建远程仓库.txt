创建仓库的时候，先进入github官网-->Sign in   进入后点击加号，选择new repository   输入一个仓库名就可以。


如果你没有本地仓库，就要选择第一个  ，如果有就选择第二个  然后刷新一下页面，他机会导入你的远程仓库中



README.md时对项目的简介。
当你已经连接远程仓库的时候，不需要提交到本地仓库了   可以直接提交到远程仓库。
git push  提交到远程仓库



当你想把远程仓库里面的 东西，复制到本地的时候。在里面有一个链接  clone and download  点击之后是一个链接
然后在你的文件夹下面进行克隆
git clone 链接
